# bbqPAINT examples

This directory contains example notebooks demonstrating the bbqPAINT analysis pipeline:

- **`1_basic_simulation.ipynb`** - Simulate fluorescence traces
- **`2_simulated_data.ipynb`** - Complete analysis pipeline from simulated data
- **`3_experimental_data.ipynb`** - Complete analysis pipeline from experimental data

Either run the examples locally in a conda environment that contains `bbqPAINT`, or open them in Google Colab:
